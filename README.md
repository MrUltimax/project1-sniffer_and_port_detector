Project: Lightweight Packet Sniffer + Port Scan Detector

Features
- Live packet sniffing
- TCP port scan detection
- Alert logging
- Desktop notifications

Requirements
- Python 3
- Scapy

Installation
pip install -r requirements.txt

Run
sudo python3 detector.py
