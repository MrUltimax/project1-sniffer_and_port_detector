# project1-sniffer_and_port_detector
This is my first python project, it contain one program for port sniffing and one for Port scan detection

I have used scapy for sniffer and port detection, project "sniffer.py" is a simple sniffing progaram and project "scan_detecter.py" is an upgraded version of sniffer.py, which works as a simple Network Intrusion detection system.

i have tested "scan_detector.py" with a nmap scan on my computer.
